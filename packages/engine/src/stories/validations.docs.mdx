# Validations
import { Story, Meta } from "@storybook/addon-docs";

<Meta title="Docs/Validations" />

To use validations, you just need to define in what event the validation will happen and relates the parameters of validation. You can use errorMessages to be shown when the validation of field were false.

There are several validations:

| Key                   | attribute type | Description       |
|-----------------------|---------|-------------------|
| [required](#required) | boolean | is required or not|
| [email](#email)| boolean | matches with email mask|
| [length](#length)| number | has this lenght |
| [greaterThan](#greaterthan) | number  | is grater than a value   |
| [hasNoExtraSpaces](#hasnoextraspaces) | boolean | has no spaces before the first word and after the last one |
| [isCreditCard](#iscreditcard)   | Array (string) | number should be a credit card number  |
| [isCreditCardAndLength](#iscreditcardandlength) | Array (string) | number should be a credit card number and should matches the length of it  |
| [isNumber](#isnumber) | boolean  | is a number  |
| [lessThan](#lessthan) | number  | number is less than another |
| [notAllowSpaces](#notallowspaces)     | boolean  | has no spaces in the field |
| [maxLength](#maxlength)     | number  | max lenght of the field |
| [notEmpty](#notempty) | boolean  | field is not empty |
| [numericRange](#numericrange)   | object (start: number end: number) | should be between two numbers |
| [onlyLetters](#onlyletters) | boolean  | not accept numbers or special chars |
| [regex](#regex)   | string  | matches a regex |
| [repeatedNumbers](#repeatednumbers)   | boolean | numbers can't repeate itself in a sequence |
| [sequentialNumber](#sequentialnumber)| boolean  | numbers can't be sequencial |
| [url](#url)   | boolean  | matches with url mask  |
| [value](#value) | string  | This key will hold masks that will run on change  |
| [isInTheList](#isinthelist) | Array (number or string)  | should belong to an array list of integers or strings|

# required 

```javascript
 validations: {
  onChange: {
    required: true,
    email: true,
  },
},
errorMessages: {
  required: 'This field is required',
},
```
